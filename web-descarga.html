Planificador de Horarios para Colegios

Herramienta sencilla para organizar horarios de profesorado, cubrir todas las franjas y ajustar limitaciones de cada docente sin necesidad de instalar nada.

Descargar aplicación Buy Me a Coffee

El botón Buy Me a Coffee está preparado con enlace provisional para añadir tu cuenta después.

Cómo funciona (explicado para profesorado)

  1. Descarga la aplicación con el botón anterior y ábrela con doble clic en tu ordenador.
  2. Define el horario del centro: hora de inicio, fin y duración de cada franja.
  3. Indica cuántos profesores necesitas en cada franja de cada día.
  4. Da de alta a cada profesor con sus horas semanales.
  5. Marca cuándo puede trabajar cada profesor y, si quieres, límite por día.
  6. Pulsa Calcular horarios para que la aplicación proponga turnos.
  7. Si faltan franjas por cubrir, ajusta disponibilidad u horas y recalcula.
  8. Usa Exportar JSON o Guardar HTML portable para compartir el resultado con otra persona o equipo.

Consejo práctico: guarda una copia semanal con fecha (por ejemplo, horario-sep-2026.html) para tener histórico.

Qué incluye la aplicación

- Alta de profesorado con horas semanales.

- Limitaciones por día y franja horaria.

- Cálculo automático de propuesta de cobertura.

- Exportación a PDF de la propuesta.

- Exportar/importar JSON y HTML portable con datos embebidos.
